# Dialogue 세팅하기

### Dialogue 이용을 위한 3단계 세팅

Dialogue를 이용하기 위해서는 아래의 3가지 단계만 거치시면 됩니다. 단계가 많아보여도, 너무 걱정하지 마세요. Dialogue팀의 Product Manager와 Engineer가 모든 과정을 도와드립니다.

* 1단계: 상담 매뉴얼 전달하기
* 2단계: 정리된 상담로직 확인하기
* 3단계: 슬랙 채널 가입하기
