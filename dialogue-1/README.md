# Dialogue로 고객 상담하기

여러 상담 채널에서 들어온 고객 문의를 복사하고, 슬랙 채널에 붙여넣기만 하세요. Dialogue가 적절한 답변을 추천해드립니다.

그리고 추천받은 답변을 골라, 상담 채널에 붙여넣으세요. 상담 매뉴얼을 스크롤하거나 검색하는 시간을 아낄 수 있습니다.
